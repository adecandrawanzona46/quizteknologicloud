MATERI PEMROGRAMAN WEB

Kerangka Web

a. html -> Berfungsi sebagai penanda awal dan penutup dokumen html.
b. head -> Berfungsi sebagai informasi page header, pada bagian head berisi informasi elemen tag title, isi index, link, script dan meta.
c. body -> Berfungsi sebagai badan atau main place untuk menempatkan konten pada website.

Komponen Padsa Web

a. p -> Tag yang berfungsi untuk membuat paragraf pada website.
b. br -> Tag yang berfungsi untuk memberikan jarak pada baris,(Break) sangat penting dalam penulisan artikel dan setiap break memiliki margin 10 pixel.
c. h1-h6 -> Berfungsi sebagai heading, yang terdiri dari 6 jenis heading.

Formating

a. (b) Berfungsi untuk menebalkan teks
b. (strong) Berfungsi untuk memberi kesan tegas pada teks
c. (emphasized) Berfungsi untuk memberikan penekanan pada teks
d. (i) Berfungsi untuk membuat teks menjadi miring
e. (small) Berfungsi untuk membuat teks menjadi kecil
f. (sub) Berfungsi untuk memposisikan teks lebih kebawah
g. (sup) Berfungsi untuk memposisikan teks lebih keatas
h. (ins) Berfungsi untuk memberi garis bawah pada teks
i. (del) Berfungsi untuk memberi garis tepat ditengah teks
j. (mark) Berfungsi untuk menandai teks
